Show <a href="http://librize.com/">Librize</a> book list by shortcode.

== Description ==

Mt8 Librize Book Listは、ショートコードでリブライズに登録されている本を表示するプラグインです。

以下のようにショートコードを記述します。

[librize-book-list id={場所ID} ]

使えるパラメータは以下の通り
id:(必須)場所ID
limit:表示件数、デフォルト:5
wrap:ラップするタグ、デフォルト：div
wrap_class:ラップするタグのクラス、デフォルト:mt8lbl_book_list
no_title:タイトルを表示しない、デフォルト：false
random:ランダム表示、デフォルト：false
link:リブライズにリンクするか、デフォルト：true
image_size:画像サイズ(縦)、デフォルト：75
